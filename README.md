# Little AI Glossary for this workshop

1. **API Key**: A unique identifier used to authenticate requests associated with your project.
1. **Agent Mode**: An AI-assisted coding mode where the AI takes a more autonomous role, making decisions and generating code based on high-level instructions.
1. **Auto-completion**: A feature in programming environments that automatically suggests code snippets, functions, or variables to complete the current line of code.
1. **Code Linter**: A tool that analyses source code to flag programming errors, bugs, stylistic errors, and suspicious constructs.
1. **Code Review**: The process of systematically examining computer source code to find and fix mistakes or improve code quality.
1. **Code Snippet**: A small region of re-usable source code, machine code, or text that can be inserted into a program.
1. **Fine-Tuning**: Adjusting a pre-trained model on a specific dataset to improve its performance on a particular task.
1. **Generative Model**: An AI model designed to create new content, such as text, images, or music.
1. **Ghost Text**: Predictive text that appears as a suggestion while coding, providing auto-completion based on the context.
1. **Hallucination**: When a generative model produces outputs that are factually incorrect or nonsensical.
1. **Inference**: The process of using a trained AI model to make predictions or generate outputs.
1. **Inline Mode**: AI assistance that provides suggestions and auto-completion at the level of individual lines of code.
1. **Plan Mode**: A mode where the AI assists with planning the structure and logic of the code before implementation, often outlining the steps or modules required.
1. **Prompt**: An input given to a generative AI model to guide its output or response.
1. **Rate Limit**: The maximum number of API requests or tokens that a user can make in a given timeframe.
1. **System Prompt**: A predefined instruction or context given to an AI model to shape its behavior.
1. **Temperature**: A parameter that controls the randomness of a model's output; higher values increase diversity and creativity.
1. **Token**: A segment of text (e.g., a word or subword) that a model processes as a single unit.
1. **Training**: The process of teaching an AI model using data to improve its performance.
1. **Word Embedding**: A representation of words as vectors in a continuous vector space, where semantically similar words are mapped to nearby points.
